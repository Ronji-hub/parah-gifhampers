## ⚙️ System Requirements

Before running this project, make sure you have:

- **Operating System:** Windows 10 or later (Mac/Linux also works)
- **Node.js:** v18 or higher — download at [nodejs.org](https://nodejs.org) *(choose LTS)*
- **npm:** Comes installed with Node.js automatically
- **Code Editor:** [VS Code](https://code.visualstudio.com) recommended

To verify your setup, open a terminal and run:
```bash
node -v
npm -v
```
Both should return version numbers. If not, see the Troubleshooting section below.

## 🚀 How to Run This Project

Follow these steps exactly in order:

**1. Clone or download this repository**
```bash
git clone https://github.com/YOUR-USERNAME/parah-gifhampers.git
```

**2. Navigate into the project folder**
```bash
cd parah-gifhampers
```

**3. Install all dependencies**
```bash
npm install
```

**4. Start the development server**
```bash
npm start
```

**5. Open your browser**  
Go to 👉 [http://localhost:3000](http://localhost:3000)

You should see the Parah Gift Hampers homepage with the headline:  
**"Thoughtfully Sourced, Beautifully Bound"**

## 🐛 Troubleshooting

| Error | Cause | Fix |
|-------|-------|-----|
| `node is not recognised` | Node.js not in PATH | Restart your computer after installing Node.js |
| `Cannot find path 'parah-gifhampers'` | Typo in folder name | Double-check spelling: `parah-gifhampers` (no 't') |
| `nmp is not recognised` | Typo in command | It's `npm` not `nmp` — letter order matters! |
| Port 3000 already in use | Another app is running | Type `Y` when asked to use a different port |
